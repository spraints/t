# T

## Installation

    $ gem install t

## Usage

    $ t start
    Starting work.
    $ t stop
    You just worked for 1 minute.
    $ t today
    You have worked for 1 minute today.
    $ t week
    You have worked for 1 minute since 2013-09-01.
    $ t path
    /Users/burke/.t.csv
    $ t edit
    (opens $EDITOR with the csv file)

Your data is stored in `$HOME/.t.csv`.
